# 好看的字体

# 英文

| Font | Description |
| --- | --- |
| Calibri | Office默认西文字体 |
| Helvetica | 最常用的非衬线字体 |
| Arial | 常用非衬线字体，Helvetica替代品 |
| Cambria | 较正式的衬线字体 |
| Times New Roman | 足够正式的衬线字体 |

# 中文

| Font | Description |
| --- | --- |
| 方正等线体 | Office默认中文字体 |
| 中易黑体 | Windows默认黑体 |
| 中易宋体 | Windows默认宋体，搭配TNR |
| 华文中宋 | 一款笔画较粗的衬线字体❤ |
