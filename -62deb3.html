Notion文章免费&自动发布到GitHub Pages

1. Notion - One workspace. Every team 1. 有一个自己的 免费网页 Github Pages 2. 在Notion中写完任何文章后的一天内 免费&自动发布到 Github Pages 中， 3. 成品请参考我的网页： 1. 首先你要有个Github账号和Notion账号，如果你是学生的话可以免费领取 GitHub Student Developer Pack 和 Notion for Education 2. 创建一个新仓库，需要给这个仓库一个比较特殊的名字，比如说 userName.github.io，注意这里的 userName 可以是任意你喜欢的名字，但是 强烈推荐使用自己的github用户名 3. 把这个 Notion表格模板 复制到自己的Notion账号中【右上角的 Duplicate 按钮】，把这个表格设置为公开【 点击右上角的 Share 按钮→ Share to web 打开 】，然后点击 Copy 按钮复制网址 3.1 比如说我复制出来的网址是 https://www.notion.so/2c4dc4affe68463da4f61f8fa93406e0 ，打开这个网址检查一下是不是对应的表格模板 4.
